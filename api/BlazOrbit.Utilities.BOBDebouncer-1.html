Class BOBDebouncer<T>

Async-aware trailing-edge debouncer. Each call to InvokeAsync(T, Func<T, Task>) resets the timer; the supplied action only runs when the specified interval elapses without further invocations.

Remarks

Designed for Blazor input scenarios — search-as-you-type, server-side validation, expensive callbacks per keystroke. Pairs with Microsoft.AspNetCore.Components.EventCallback<TValue>: pass the callback's InvokeAsync as the action.

The debouncer is System.IDisposable; consume from a component's DisposeAsync/Dispose path so a pending invocation does not fire after the component is unmounted.
