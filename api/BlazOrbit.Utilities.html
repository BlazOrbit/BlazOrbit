BOBAsyncHelper

Helper for fire-and-forget async operations in Blazor UI contexts.

Use SafeFireAndForget(Func<Task>, ILogger?) when an async operation must be started from a synchronous callback (e.g. an event Action? handler) where awaiting is impossible. The helper suppresses the four canonical teardown exceptions and logs anything unexpected so it does not end up in System.Threading.Tasks.TaskScheduler.UnobservedTaskException.

Prefer awaiting directly whenever the calling context is already async; this helper is a last resort for sync→async bridges.

BOBDebouncer<T>

Async-aware trailing-edge debouncer. Each call to InvokeAsync(T, Func<T, Task>) resets the timer; the supplied action only runs when the specified interval elapses without further invocations.

BOBThrottler<T>

Async-aware leading-edge throttler. The first InvokeAsync(T, Func<T, Task>) call within an idle window runs immediately; subsequent calls inside the configured interval are coalesced — only the latest argument is kept and executed once the interval elapses.
